About the Project
------------------------------------------
The goal of this project is to create a webapp that allows anyone who might benefit from reading, researching, or reporting on data center policy to easily and intuitively search for and find data center policy relevant to their needs.
------------------------------------------



Features
------------------------------------------
The primary feature of this webapp is the database and its search tools that will allow users to find data center policy. Additionally, the webapp will include a blog that will allow for the documentation of data center related events that might not necessarily fall under the explicit category of policy, and for the highlighting of important new policy. Example blogs might be about local resistance to the construction of a new data center, the passing of new federal data center policy, or opinion pieces about data center policy.
------------------------------------------



Tech Stack
------------------------------------------
Frontend - React, HTML, CSS
Backend - Django, REST framework, Wagtail CMS
Database - PostgreSQL

Web Hosting - DigitalOcean App Platform
Database Hosting - DigitalOcean PostgreSQL
